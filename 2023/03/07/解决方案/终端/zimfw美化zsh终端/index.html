zsh安装

  • 查看自己系统是否存在zsh

  • 查看是否是使用的 zsh

  • 切换为 zsh（macos 自带默认是 zsh，如果不是进行切换即可）

  • ubuntu

zim安装

  1. 如果你安装过卸载 ohmyzsh

  2. 安装zimfw

  • 如果有这个错，请把 DNS 改成8.8.8.8

  • 等待一会儿，可能需要 fq（github）

  • zimfw安装完成

插件和主题配置

git clone [https://github.com/matyle/matyle.github.io](https://github.com/matyle/matyle.github.io)
